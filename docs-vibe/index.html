🏗️ Kiến trúc Runtime 4 Tầng

Kiến trúc phân tầng theo Microsoft Agent Mode. Giao tiếp qua contract — không gọi trực tiếp giữa các tầng.

T1 — INTERFACE (Giao diện)

Validate input, classify prompt, format output. Cửa tiếp nhận mọi yêu cầu từ người dùng.

Agent: pxh-help — State: INTERFACE

Ở lại bàn làm việc cho đến khi TUI kết thúc. Truyền tín hiệu dashed → CEO PXH.

T2 — ORCHESTRATION (Điều phối)

Auto-route task, track state, enforce retry/recovery/reflection. Bộ não trung tâm.

Agent: pxh-pm — State: ORCHESTRATION

Ở lại bàn làm việc cho đến khi TUI kết thúc. Truyền tín hiệu dashed → tất cả T3 agents.

T3 — WORKERS (Nhân công)

Thực thi domain: code, test, fix bug, review, build, UI/UX. 7 agents chuyên biệt.

Ngồi vào bàn khi được gọi, rời bàn khi xong việc (8s idle). Truyền tín hiệu dashed về CEO + đồng nghiệp.

T4 — INFRASTRUCTURE (Hạ tầng)

Persist state, checkpoint, log, alerting. Ghi nhận mọi sự kiện, không quyết định.

Agent: pxh-save-history — State: INFRASTRUCTURE

Rời bàn khi xong việc. Truyền tín hiệu dashed → CEO PXH.

🏢 Văn Phòng Ảo

Webview 2D Open Office — 10 nhân vật pixel-art trong văn phòng mở single-floor, real-time sync với TUI qua SSE.

Khởi động

# ON — Start server + mở browser
.\pxh-office-on.bat

# OFF — Tắt server
.\pxh-office-on.bat off

# RESTART — Tắt + start mới
.\pxh-office-on.bat restart

# Hoặc chạy tay
node skills/virtual-office/templates/server.mjs
start http://localhost:2910

Sync real-time với TUI

.\skills\virtual-office\templates\hook-opencode.ps1 "your prompt"

⚡ Real-time Sync

Cơ chế đồng bộ thời gian thực giữa TUI (terminal) và webview (browser):

Event Flow

TUI (opencode)
  │
  ├── hook-opencode.ps1 ──→ _shared/opencode-state.json ──→ server.mjs (watchFile)
  │                                                              │
  ├── file changes ──→ office-bridge.mjs ──→ emit() ──→ broadcast() (SSE)
  │                                                              │
  └── HTTP POST /emit /state ─────────────────────────────→ SSE → browser

Bridge — Auto-detect workspace activity
